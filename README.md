# GoTo Gojek Tokopedia - FP&A Analysis FY2025

**Independent Financial Planning & Analysis Study**  
PT GoTo Gojek Tokopedia Tbk (GOTO.JK) | FY2023–2025 Actuals + FY2026 Forecast  
Prepared by: Syahraini | June 2026

---

## Overview

This project presents a comprehensive FP&A analysis of GoTo's FY2025 
audited financial statements, covering historical performance analysis 
and a forward-looking scenario forecast for FY2026.

The analysis was conducted independently using publicly available 
consolidated financial statements (Laporan Keuangan Konsolidasian 
31 Desember 2025).

---

## Key Findings

- Revenue grew **+15.3% YoY** to Rp18.3T while total costs grew only 
  **+3.1%**, reflecting strong operating leverage
- Operating loss narrowed **83.0%** from Rp2.2T to Rp378M; GoTo is now 
  only **2.1% away from operational breakeven**
- OCF turned **positive for the first time** at +Rp307M (vs -Rp622M FY2024)
- FY2026 Base Case projects **first-ever net profit of +Rp951M**

- `COVER` — Model index and color coding convention

- `IS` — Income Statement FY2023–2025 actuals + FY2026 forecast
- `BS` — Balance Sheet FY2023–2025 with variance analysis
- `CF` — Cash Flow Statement FY2023–2025 with FCF calculation
- `SEGMENT` — Revenue breakdown by business segment
- `KPI` — Summary dashboard with trend indicators

**Forecast Methodology:**
- Driver-based revenue forecasting per segment
- OCF projected using FY2024–2025 average conversion ratio (62.3%)
- CAPEX assumptions: Bear 5.5%, Base 6.0%, Bull 6.5% of revenue
- Tax assumption: zero effective rate due to accumulated fiscal losses

## Key Analytical Insights

**1. Structural Break — Tokopedia Deconsolidation**  
TikTok's entry as Tokopedia shareholder in FY2024 changed GoTo's 
consolidation structure. Tokopedia moved from full consolidation to 
equity method (PSAK 228), creating non-apple-to-apple comparability 
for Revenue segments between FY2023–2024.

**2. SBC Normalization**  
Share-based compensation fell 57% (Rp1.37T → Rp593M). As a non-cash 
item, this improves reported P&L without reflecting operational cash 
efficiency. Forecast assumes SBC remains at FY2025 levels.

**3. Lending Credit Quality**  
Financing receivables grew +72.6% YoY. Coverage ratio for overdue 
>30 days: 89.0% (very conservative). Total coverage: 13.0%. 
Overdue ratio increased from 7.8% (FY2024) to 10.2% (FY2025) — 
within acceptable range but requires monitoring.
